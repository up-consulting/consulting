# 오늘일기 — 운영·사용 가이드

30일 코칭형 습관 케어 웹앱. 코치가 멤버를 30일간 따라가며, 멤버는 매일 미션을 인증하고 기분 일기를 남깁니다.

---

## 0. 한눈에 보는 구조

- **정적 웹앱**(빌드 없음, 순수 HTML/CSS/JS) + **Supabase**(DB·인증·스토리지)
- 배포: GitHub `up-consulting/consulting` → Vercel 자동 배포 (푸시하면 자동 반영)
- 역할: `member`(멤버) · `coach`(코치) · `admin`(운영자)

| 파일 | 화면 |
|------|------|
| index / signup / login / setup | 진입·가입·로그인·초기설정 |
| home / tracker / journal / goals / profile | 멤버 화면 |
| admin | 코치 패널 (대시보드·검토·멤버연결·멤버상세) |

---

## 1. 최초 1회 세팅 (Supabase SQL Editor)

순서대로 **한 번씩** 실행:

1. `supabase/schema.sql` — 테이블·RLS·트리거·스토리지
2. `supabase/lifecycle.sql` — **매일 자정(KST) 자동 처리** (미제출→놓침, 잠금→오늘, 30일 완료, 알림 생성)
   - ⚠️ `pg_cron` 확장이 필요합니다. Supabase → Database → Extensions 에서 **pg_cron** 활성화 후 실행
3. `supabase/migrate_roles.sql` — 멤버/코치 분리 가입용 컬럼·트리거

> lifecycle.sql 을 안 돌리면 **날짜가 자동으로 안 넘어가서** 멤버가 DAY 1에 멈춥니다. 테스트 시 수동 진행: `select advance_day();`

### 이메일 인증 / 리다이렉트
- Supabase → Authentication → Providers → Email: 테스트 중엔 "Confirm email" **꺼두면** 가입 즉시 로그인됩니다.
- Authentication → URL Configuration 의 **Site URL / Redirect URLs** 에 배포된 Vercel 도메인을 추가.

---

## 2. 코치 계정 만들기

코치는 가입 화면에서 **"코치로 시작"** 을 선택해 가입합니다. (migrate_roles.sql 실행 후부터 자동으로 코치 역할 부여)

이미 만든 계정을 코치로 승격하려면:
```sql
update profiles set role='coach', status='approved' where email='코치이메일';
```

---

## 3. 멤버 플로우 (member)

1. **가입** — "멤버로 시작" 선택 + **담당 코치 이름** 입력(코치의 가입 이름과 정확히 일치해야 함)
2. **대기** — 코치가 연결하기 전까지 홈은 "코치 연결 대기" 상태
3. **연결 후 30일 시작** — 매일:
   - **홈**: 오늘의 미션(=코치가 정한 메인 목표) **인증하기**(사진+메모) → 검토 대기
   - **기분 기록**: 1~5점 + 한 줄
   - **일지**: 누적 기록(미션 인증·느낀점·코치 피드백) 모아보기
   - **목표**: 서브 목표 추가/수정/삭제 + 진척 기록
   - **마이**: 내 정보·코치·요약·다짐 문구·로그아웃

---

## 4. 코치 플로우 (coach) — `admin.html`

- **대시보드**: 함께하는 멤버 수, 살펴볼 신호(연속 미제출/기분 하락), 멤버 목록
- **검토**: 멤버가 올린 인증 사진·메모 확인 → **승인/반려 + 코멘트**(피드백)
- **멤버 연결**: 나를 담당 코치로 지정한 대기 멤버를 연결(메인 목표 입력)
- **멤버 상세**(멤버 행 클릭): 메인 목표 수정 · 이행률 · 30일 트래커 · 기분 추이 · 서브 목표 · **인증/일기/피드백 타임라인** · 코치 노트

---

## 5. 핵심 개념

- **메인 목표 = 오늘의 미션**: 코치가 연결 시(또는 멤버 상세에서) 설정/수정. 멤버 홈에 표시.
- **서브 목표**: 멤버가 직접 관리하는 작은 목표들 (목표 탭).
- **인증(daily_tasks)**: 하루 1건, 사진/메모 제출 → 코치 검토(승인/반려) → 피드백.
- **일기(mood_logs)**: 하루 1건, 기분·에너지·메모.
