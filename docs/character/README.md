# 캐릭터(오늘이) 교체 가이드

새로 만든 캐릭터를 여기에 넣으면 **가이드북 + 앱(홈·리캡·성장가이드)**에 반영해 드립니다.

## 넣어주실 파일 (PNG, 정사각형 권장 · 배경 투명)

성장 단계별 7종 + 시듦 1종:

| 파일명 | 단계 | 조건 |
|--------|------|------|
| `stage0.png` | 씨앗 | 시작 전 |
| `stage1.png` | 떡잎 | 미션 1~2개 |
| `stage2.png` | 새싹 | 미션 3~6개 |
| `stage3.png` | 쑥쑥 | 미션 7~13개 |
| `stage4.png` | 꽃봉오리 | 미션 14~21개 |
| `stage5.png` | 활짝 | 미션 22~29개 |
| `stage6.png` | 만개 | 30개 완주 |
| `wilt.png` | 시듦 | 연속 미제출 |

> 단계 수를 줄이거나(예: 4단계) 다른 컨셉으로 바꿔도 됩니다. 파일만 주시면 앱의 단계 기준과 화면 배치를 맞춰 드립니다.
