연동 설정이 필요해요

Supabase 프로젝트를 연결하면 실제로 동작합니다.

1. 새 Supabase 프로젝트 만들기

supabase.com → New project 생성

2. SQL 실행

SQL Editor 에 supabase/schema.sql → supabase/lifecycle.sql 순서로 붙여넣고 실행

3. 키 입력

Project Settings → API 의 URL · anon key 를 assets/supabase-client.js 상단 두 줄에 붙여넣기

4. 이메일 인증 끄기(테스트용)

Authentication → Sign In/Up → Email 의 "Confirm email" 끄면 가입 즉시 로그인됩니다.
